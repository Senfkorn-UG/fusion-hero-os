# Heroic Core Foundation — Layer 0 (Immutable)

**Status**: Immutable. Changes require extraordinary justification, documented multi-party audit, and explicit human ratification. Reversion must remain trivial.

This document is the single source of truth for the non-negotiable base of all heroic work.

---

## 1. Core Postulates (Layer 0)

1. **Epistemic Hygiene is non-negotiable**
   - No metaphor may be used as proof.
   - No model may be silently promoted to "Satz" (theorem).
   - Modal language ("could", "might", "suggests") must not collapse into declarative certainty without additional evidence.

2. **Geltungskategorien Discipline (Mandatory Labeling)**
   Every formal or truth-claiming statement **must** be explicitly labeled with one of:

   | Category    | Meaning                                      | Proof burden          | Upgrade path                     |
   |-------------|----------------------------------------------|-----------------------|----------------------------------|
   | **Satz**    | Proven, with traceable complete proof or citable source | Highest              | None (already maximal)          |
   | **Bedingt** | Holds conditionally under explicitly stated assumptions | High (assumptions must be named) | Only by relaxing/falsifying assumptions |
   | **Modell**  | Structural analogy or heuristic simplification | None (illustrative)  | Never auto-promoted             |
   | **Axiomatisch** | Deliberate postulate / definition       | None (by fiat)       | Never presented as derived      |
   | **Fragment**| Incomplete or provisional                    | Explicitly open      | Can mature only with new evidence |

   **Cardinal Rule**: An Axiomatisch or Modell statement must **never** be presented as a Satz without an explicit, reviewed re-derivation.

3. **Self-Criticism as Default Operation**
   - Every non-trivial output is subject to Critical Meta-Analysis.
   - Patterns to detect and correct: Entkategorisierung, Modal-Kollaps, Selbst-Mythologisierung, Monumentalisierung, Metapher-als-Beweis.
   - The 5-Dimensions Review (Evidenz, Logik, Alternativen, Implikationen, Unsicherheiten) is the mandatory minimal gate.

4. **Radical Nothing-Bereitschaft (Openness to Annihilation)**
   - No construct, identity, or prior conclusion is protected from revision.
   - The only protected elements are the Layer 0 rules themselves.
   - "Nothing" is treated as a valid and sometimes optimal state.

5. **Primacy of Traceability & Evidence**
   - Every substantive claim must be traceable to primary source, explicit reasoning step, or declared axiom.
   - Internal consistency across modules is required; contradictions must be surfaced and resolved or clearly scoped.

6. **Reversibility**
   - All Layer 1+ artifacts (code, documents, archives, processes) must be versioned such that any prior state can be restored.
   - Layer 0 itself has a higher bar: any proposed change must include a rollback plan and a simulation of long-term effects.

7. **Human Ratification for Foundation Changes**
   - Automated or AI-proposed modifications to Layer 0 are **invalid** until a human explicitly accepts them after reading the full audit.

---

## 2. Enforcement Principles

- **Default posture**: Skeptical, precise, category-explicit.
- **Language rule**: When in doubt, mark lower (Modell / Fragment) rather than higher.
- **Scope rule**: Claims outside the current scope are not smuggled into conclusions.
- **Failure mode**: Any detected silent upgrade or unlabelled model triggers an immediate corrective pass before proceeding.

---

## 3. Relationship to Higher Layers

- **Layer 1 (Core Modules)**: Implements operational tools (5-D review, Erkenntnisprozess, archiving, etc.). All must import and respect this foundation.
- **Layer 2+**: Concrete operations and meta-processes. May propose improvements to their own layer but must route Layer 0 impacts through extraordinary review.

This foundation is the ground on which the heroic methodology stands. Everything else is scaffolding that can be rebuilt.

---

## 4. Minimal Checklist for Any Heroic Artifact

Before considering an output "ready":

- [ ] Every truth-claim carries an explicit Geltungskategorie.
- [ ] No metaphors presented as derivations.
- [ ] 5-Dimensions analysis performed or referenced.
- [ ] Open uncertainties and fragments listed.
- [ ] Trace to source/axiom/step present.
- [ ] Version / reversibility path recorded.

Failure on any item means the artifact has not yet passed the foundation gate.

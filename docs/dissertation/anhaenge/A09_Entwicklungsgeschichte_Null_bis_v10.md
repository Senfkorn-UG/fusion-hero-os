## Synthese

Entwicklung ist selbst ein Sisyphos-Zyklus: Ruf → Schwelle → Prüfungen → Abgrund → Wandlung → Rückkehr. Diese Geschichte ist **kein** Mythos statt Beweis, sondern die **Genealogie der Organe**, damit man versteht, warum Module existieren.

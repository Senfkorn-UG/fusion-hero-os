## Rules for Creating New Chains (v7.5)

1. Start with Δ (Distinction) unless continuing an existing frame.
2. Respect the official `dependency_table` (no invalid jumps).
3. End with Ψ only when a real Self-Binding / identity update occurs.
4. Use Χ (Distance) generously for regulation and sustainability.
5. Every chain should be testable in real life within a defined time box.
